# Real-Time Head Tracking Camera Orientation System

## 📌 Overview
This project presents a real-time head-tracking based camera control system for FPV drones. The system allows the user to control camera orientation using head movements without rotating the entire drone.

---

## 🚨 Problem Statement
In conventional FPV systems, camera orientation is controlled by rotating the entire drone. This results in:
- Excessive power consumption
- Unstable video output
- Unnecessary mechanical movement

---

## 💡 Proposed Solution
We propose a system where:
- Head movement is captured using an IMU sensor
- Data is processed using ESP32
- Commands are transmitted wirelessly
- A gimbal system rotates the camera independently

This reduces power usage and improves stability.
